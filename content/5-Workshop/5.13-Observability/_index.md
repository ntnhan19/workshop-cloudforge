### Observability Overview

A Production system cannot lack "eyes" to monitor its health and debug issues. AWS provides a robust toolset to ensure comprehensive Observability.

The focus of this chapter includes:
1. **Amazon CloudWatch:** Centralized log management from ECS Containers, Lambda, and RDS. Set up Alarms for high CPU usage or 500 errors.
2. **AWS X-Ray:** Distributed Tracing to know exactly how many milliseconds a Request takes at each stage, from entering the API Gateway to querying the Database.
